"Do They Use?" (DTU) service demo page with remote frames

<iframe> with remote your_app.html

In this left iframe is displayed example of "your app" — some "monitored app" with links, input fields and buttons. These elements have "data-dtu" attributes. And the frame itself imports DTU JavaScript SDK — everything is set according to this setup instruction.

When you interact with the elements below, usage stats (via window.localStorage of this browser) is being saved and becomes available to the right frame — to the DTU analytics app.

<iframe> with remote dtu_app.html

In this right iframe is displayed DTU analytics app that has access (via window.localStorage of this browser) to the data generated by the app in the left iframe.

Interact with the elements in the left frame and then find reports for these interactions in this right frame. This is the concept of how this "elements usage analytics" works. If you want to see how DTU analytics app displays data for lots of reports, please visit Cloud-based demo page.
