How this demo works

Every interaction element (button, link, menu, drop-down, date-time picker, etc.) on a whole https://dotheyuse.com website (including this web page as well) has its own unique and meaningful "data-dtu" attribute:

Every page also imports and inits Do They Use service JavaScript SDK with custom "topic": "real usage":

This setup allows to track all your interactions with any tagged (we want to be monitored) element on this website. And all these interactions appear under "topic": "real usage" in demo account:

By default chart displays "all available" information: stats for all the elements from the whole web site (including demo account itself) at any time. But you can filter for any particular element at any time period. If an element has never had interactions or it has no "data-dtu" attribute assigned, then it will not be available for filtering.

The more you interact with different elements on website — the more of them appear in stats under "topic": "real usage".
